Sobre o DOSPRN

Este programa foi projetado para suportar uma impressão de programa DOS em tinta moderna impressoras a jato e laser. Muitas impressoras baratas não podem imprimir em modo de texto, mas imprima apenas no modo gráfico (os chamados dispositivos GDI). Além disso, programas DOS antigos (e alguns mais novos) foram criados no de uma impressora matricial em modo texto. Além disso, muitas impressoras não suportam os idiomas nativos no modo texto.

O DOSPRN ajuda a superar esse problema. Você pode usá-lo com DOS e personalize qualquer impressora. Você deve simplesmente instalar o DOSPRN, executar seu programa DOS e imprimir relatórios.

O DOSPRN é distribuído como Shareware. É possível aprenda a usar o DOSPRN durante o período de teste. Após esse período, você deve registrar sua cópia ou removê-la completamente do seu computador com o utilitário de desinstalação no Grupo de Programas DOSPRN.

Instalação

Baixe e execute o arquivo de extração automática DOSprn200.exe. Você receberá algumas perguntas simples. Durante a instalação, O DOSPRN copia todos os arquivos e atalhos necessários para o seu computador.

Início do programa

Você pode iniciar o DOSPRN clicando em um atalho em sua área de trabalho ou na pasta "Inicialização" (caso você tenha selecionado esta opção durante a instalação). Depois de iniciar o DOSPRN, você pode encontrar seu ícone na área da bandeja (próximo ao relógio do sistema, geralmente no canto inferior direito). DOSPRN pronto para trabalhar agora.

Recursos do programa

Se você clicar com o botão direito do mouse no ícone DOSPRN na bandeja, você verá um menu pop-up. Este menu dá acesso às principais opções do programa. Menu Opções:

  • Abrir - abre a janela de propriedades principal 
  • Imprimir arquivo - imprime um arquivo
  • Trabalho de impressão - imprime seu trabalho atual (se o 'auto imprimir' está desativada)
  • Orientação - altera a orientação do papel
  • Limpar trabalho - remove sua tarefa atual do DOSPRN fila (se a opção 'impressão automática' estiver desativada)
  • Cancelar - fecha o menu pop-up
  • Saída - fecha o DOSPRN

Janela Propriedades principais

Tempo limite do trabalho

 Esta é a quantidade de tempo de inatividade antes da impressão Começa. Se o aplicativo DOS não enviar nada para a impressora durante esse período, O DOSprn começa a imprimir.

Impressão automática

 Se esta opção estiver ativada, o DOSPRN inicia automaticamente Imprimindo seu trabalho atual após o período de tempo limite. Caso contrário, você pode Configure seu trabalho de impressão manualmente.

Porta de captura

 O DOSPRN captura todos os dados que os aplicativos enviam para o LPT com tradução completa de dados de impressão.

Imprimir arquivo

 Você pode simplesmente enviar um arquivo de texto em ASCII para o seu impressora.

Trabalho de impressão

 Imprima seu trabalho atual (se o 'auto imprimir' está desativada)

Limpar trabalho

 Remova seu trabalho atual da lista de tarefas (se o 'auto imprimir' está desativada)

Nome do arquivo de spool (máscara de arquivo)

 Nome do arquivo (ou máscara de arquivos) que contém o arquivo temporário dados de buffer de spool.

Nota: se você tiver que especificar um valor não padrão para esta opção, você deve desative o recurso 'Porta de captura'.

Idioma da interface:

 Selecione o idioma da interface do programa. Inglês, Francês, Deutsch, Swedish, Dutch, Italiano, Slovak, Español, Português, Grego, Romeno, Húngaro, Catalão, Croata, Russo, Ucraniano, Bielorrusso, Interfaces letã, checa, búlgara, turca e polaca já estão disponíveis. Você pode traduzir o componentes da interface em seu idioma nativo. Por favor, leia mais sobre o idioma módulos de interface ou suporte de e-mail.

Orientação

 Selecione a orientação padrão do papel

Enrolar linhas longas

 Quando esta opção é selecionada, o DOSPRN irá envolver todos os linhas que excedam os limites especificados

Usar símbolo de feed de formulário

 O DOSPRN pode processar o símbolo FormFeed (ASCII 12) ou ignorá-lo.

LF como CR/LF

 O DOSPRN pode processar o símbolo LineFeed (ASCII 10) como um retorno simples para a posição de impressão atual para a mesma posição na próxima linha, ou como par CarriageReturn (ASCII 13) e LineFeed (ASCII 10). No segundo caso, a impressão position se moverá para o início da próxima linha.

Execução minimizada

 Se você selecionar esta opção, o DOSprn começará minimizado na bandeja do sistema por padrão. Caso contrário, a janela de propriedades principais será aberto após o início.

Impressora

 Selecione sua impressora atual. Nota: se você usar um impressora de rede, você não precisa especificar uma porta local

Emulação de sequências de escape

 O programa possui suporte a sequências ESC. O DOSPRN pode emular vários esquemas, incluindo os frequentemente usados Padrão Epson Esc-P2, HP PCL, IBM Proprinter, o esquema DOSPRN padrão ou um esquema definido pelo usuário.

Propriedades das fontes

Fontes

 Você pode ver a lista de fontes instaladas aqui. Monoespaçado (largura fixa) As fontes são exibidas em verde, enquanto as fontes proporcionais estão em vermelho. Recomendamos que você use fontes monoespaçadas (como "Courier New" ou "Lucida Console").

Exibir somente monoespaçamento

 Se esta opção for selecionada, a lista de fontes incluirá somente fontes monoespaçadas

Emulação de página de código

 Página de código. DOSPRN suporta 850 (Western European), 852 (Central European), 737 (Greece), 874 (Thai), 932 (Japan [shift_jis]), 936 (Simplified Chinese [gb2312]), 949 (Korean [EUC-KR]), 950 (Traditional Chinese [Big5]), 437 (USA), 862 (Hebrew), 775 (Baltic), 855 (Cyrillic), 1125 (Ukrainian), 866 (Russian) e outras páginas de código agora

O botão "Visualizar" permite visualizar o arquivo ASCII.tbl nesta página de código. A imagem de visualização deve corresponder à aparência deste arquivo na caixa do DOS.

Se você se sente confortável com tudo isso, pode ler mais Sobre os módulos do CodePage estruture e modifique os arquivos do CodePage por conta própria. Caso contrário, você pode enviar um e-mail para o suporte se tiver problemas ou necessidades especiais.

Tamanho

 Um tamanho de fonte padrão.

CPI

 Este é um parâmetro que especifica o número de caracteres por polegada. Há interdependência no estilo e tamanho da fonte.

LPI

 Este é um parâmetro que especifica o número de linhas por polegada. Há interdependência no estilo e tamanho da fonte.

Margens

 Os tamanhos das margens superior, esquerda, inferior e direita serão aplicados ao imprimir um documento de texto de Aplicativos DOS. Não há interdependência nas configurações do documento DOS.

Visualizar

 A seção de visualização exibe a fonte selecionada no momento no modo WYSIWYG
